A lightweight coroutine-native flow runtime for Lua

Orbit is a lightweight, single-file Lua library for building and running stateful workflows (flows) using a node-based approach. It supports conditional transitions, retries, batch processing, and is native to Lua coroutines for asynchronous execution.
