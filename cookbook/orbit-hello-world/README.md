# orbit Hello World (Lua)

Your first orbit application in Lua! This simple example demonstrates how to create a basic orbit app using the Lua runtime.

## Project Structure

```
.
├── utils/         # Utility functions
├── flow.lua       # orbit implementation
├── main.lua       # Main application entry point
└── README.md      # Project documentation
```

## Setup

1. Make sure you have Lua installed.
2. Ensure `orbit.lua` is in the parent directory or in your `LUA_PATH`.
3. Run the example:
```bash
lua main.lua
```

## What This Example Demonstrates

- How to create your first orbit application in Lua.
- Basic orbit concepts and usage.
- Simple example of orbit's capabilities.
