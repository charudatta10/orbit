# orbit Hello World

Your first orbit application! This simple example demonstrates how to create a basic orbit app from scratch.

## Project Structure

```
.
├── docs/          # Documentation files
├── utils/         # Utility functions
├── flow.py        # orbit implementation
├── main.py        # Main application entry point
└── README.md      # Project documentation
```

## Setup

1. Create a virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the example:
```bash
python main.py
```

- How to create your first orbit application
- Basic orbit concepts and usage
- Simple example of orbit's capabilities
