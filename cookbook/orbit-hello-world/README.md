# PocketFlow Hello World (Lua)

Your first PocketFlow application in Lua! This simple example demonstrates how to create a basic PocketFlow app using the Lua runtime.

## Project Structure

```
.
├── utils/         # Utility functions
├── flow.lua       # PocketFlow implementation
├── main.lua       # Main application entry point
└── README.md      # Project documentation
```

## Setup

1. Make sure you have Lua installed.
2. Ensure `pocketflow.lua` is in the parent directory or in your `LUA_PATH`.
3. Run the example:
```bash
lua main.lua
```

## What This Example Demonstrates

- How to create your first PocketFlow application in Lua.
- Basic PocketFlow concepts and usage.
- Simple example of PocketFlow's capabilities.
